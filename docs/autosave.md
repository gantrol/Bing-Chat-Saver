# Auto save

## 场景

进入方式不同：

1. 通过点击“必应聊天”按钮进入聊天页面
2. 进入必应搜索，点击聊天按钮，进入聊天页面

使用过程：

1. 一直使用，直到关闭标签页或浏览器
2. 直到页面刷新
3. 使用过程中，鼠标滚轮下滑，退出聊天页面
4. 退出聊天页面后回到聊天页面
